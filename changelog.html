Changelog

As a developer I really like tracking my activities. This section is a changelog of website structure. It can be useful for people interested in developing a similar project. If you are interested in source code refer to the GitHub repository.

1.1 2022-02-05

Switch to local Jekyll

  • citation and bibliography with ieee style [1] [2]
  • jekyll-scholar plugin [3]
  • The full list of Emoji codes [4]
  • jemoji plugin [5]
  • GitHub Pages cannot build sites using unsupported plugins. If you want to use unsupported plugins, generate your site locally and then push your site’s static files to GitHub [6]

1.0 2021-06-20

First useful version of website

  • Google Analytics setup for Jekyll [7]
  • Fixed a problem in resume dropdown menu using Liquid Filter “remove” and Operator “contains” [8]

Navbar fixing and resume page simplified

Categorize posts with a tag
